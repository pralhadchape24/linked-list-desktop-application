## 📖 Overview

Linked List Visualizer is a desktop application developed using **C++** and **Qt** to demonstrate the working of linked list data structures through an interactive graphical interface.

It allows users to perform linked list operations while visualizing changes in real time, making it useful for learning data structures.

---

## ✨ Features

- Insert Node
- Delete Node
- Search Node
- Display Linked List
- Interactive GUI
- Real-time Visualization
- Dynamic Memory Management
- Object-Oriented Design
